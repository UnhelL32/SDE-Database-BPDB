# SDE-Database-BPDB
A app to view current SDE seniority list with the following functions
​1. Gateway & Authentication Layer
​Secure Access Portal: Features a dedicated, minimalist landing page requiring a specific User ID (SDE) and Password (2026) to access the internal system.
​Professional Branding: Visually credits you, Md. Minhajul Haque (Sub Divisional Engineer, BPDB), as the system creator directly on the entry gateway and across the application footer.
​2. Directory Database Interface
​Full Data Storage: Pre-loaded with your master dataset containing comprehensive registry files compiled from the Seniority List of SDE .pdf.
​Dynamic Search Engine: Real-time filtering capable of instantly matching keywords against an officer's Name, ID Code, Designation, or Serial Number.
​Complete CRUD Pipeline: Includes sleek, modal-driven forms to dynamically Add new engineer records, Edit/Update existing entry parameters, or Delete retired entries with persistent session saving.
​3. Analytics & Visualization Hub
​Post Segregation Counters: Automatically categorizes and counts current engineers based on their official posts (e.g., SDE, XEN, Deputy Managers, Directors) into responsive top summary cards.
​PRL Year Distribution Graph: Uses Chart.js to map out a sleek bar chart visualizing the volume of upcoming engineer retirements by year, making future manpower planning highly visible and scannable.
